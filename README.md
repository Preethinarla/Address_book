# Address_book 

It is a RESTful Address Book API built using FastAPI.
It allows users to create, update,delete, and retrieve address with geolocation Support.

This api supports fetching nearby addresses based on coordinates and distance.


----Features ---- 
- Create a new address
- Update existing address
- Delete address
- Get all addresses
- Get address by ID
- Fetch nearby addresses using latitude, longitude, and radius
- Input validation using Pydantic
